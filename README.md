# Modern Kişisel Portfolyo Sitesi

Bu proje, şık ve modern bir kişisel portfolyo web sitesidir. GitHub Pages üzerinde kolayca yayına alınabilir.

## Özellikler
- **Responsive Tasarım:** Mobil, tablet ve masaüstü cihazlarla tam uyumlu.
- **Karanlık Mod:** Göz yormayan gece modu seçeneği.
- **Smooth Scroll:** Bölümler arası yumuşak geçiş.
- **Modern Stil:** Temiz ve minimalist arayüz.

## Nasıl Yayınlanır? (GitHub Pages)
Bu siteyi canlıya almak için şu adımları izleyin:
1. GitHub deponuzun ana sayfasına gidin.
2. **Settings** sekmesine tıklayın.
3. Sol menüden **Pages** seçeneğini bulun.
4. **Build and deployment** kısmında, Branch olarak `main` ve klasör olarak `/ (root)` seçin.
5. **Save** butonuna tıklayın.
6. Birkaç dakika içinde siteniz `https://kullaniciadi.github.io/modern-portfolio-site/` adresinde yayında olacaktır.

## Özelleştirme
- `index.html` dosyasındaki metinleri ve bağlantıları kendinize göre güncelleyin.
- `style.css` dosyasından renkleri (`--primary-color`) değiştirebilirsiniz.
